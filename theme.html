Using non-supported configurations may cause visual errors not intended by the designer. Sometimes extra configurations are supported through the use of mods.

How to install list CSS.

  1. Navigate to your style settings on MyAnimeList.

    If your page does not look like this, double check you have the "Modern" option selected. This option is found near the top, where it says "Switch Template".

  2. Open your currently selected style by clicking on it. The selected style will be highlighted on your page.

  5. In the sidebar, navigate down to the "Add Custom CSS" area. Click to expand it if necessary.

  6. Copy the CSS from the theme customiser and paste it into the "Add Custom CSS" text box on MyAnimeList.

  7. Click "Save" and your theme should be applied!

  8. If this guide did not work for you or you have further questions, please check the theme-specific help page provided by the theme author. Get help.

How to install classic list CSS.

  1. Navigate to your style settings on MyAnimeList.

    If your page does not look like this, double check you have the "Classic" option selected. This option is found near the top, where it says "Switch Template".

  2. Open the "Advanced CSS List Design" link. On this page, open either style.

  3. Copy the CSS from the theme customiser and paste it into the text box on MyAnimeList.

  4. Click "Update CSS" to apply your changes.

  5. Go back to your style settings, then to "Change My List Style".

    On this page, double check you have the same style number selected as the one you copied the CSS to. If they mismatch, change it to the correct number and click "Submit".

    And with that, you should be done! Check the changes on your anime or manga list.

  6. If this guide did not work for you or you have further questions, please check the theme-specific help page provided by the theme author. Get help.
